How does weather change with latitude?

In this project, I created a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, I used a Python library, Citipi - (https://pypi.python.org/pypi/citipy), the OpenWeatherMap API - (https://openweathermap.org/api), and a little common sense to create a representative model of weather across world cities. A series of scatter plots showcases the relationships between latitude and maximum temperature, humidity, cloudiness, and wind speed for over 500 randomly selected cities on February 8, 2021.
